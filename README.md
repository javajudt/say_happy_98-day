## Usage

Use `{}` markup to highlight a part of text with a different color.

```bash
./make-pdf.rb -t 'print("{Happy birthday, %username%!} You are awesome!")'
              -b '0D4F8B'
              -c 'E3170D'
              -o 'postcard.pdf'
```

Will generate you the following [postcard](examples/postcard.pdf).
